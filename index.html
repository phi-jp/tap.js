tap.js

画面をタップするとエフェクトが出るよ

Github Demo


Usage

<script src='http://cdn.rawgit.com/phi-jp/tap.js/v0.1.0/tap.js'></script>
<script>
// only
tap();
</script>

Options

// 指定可能なオプション
tap.options.color     = 'red';  // エフェクトの色
tap.options.radius    = 40;     // エフェクトの半径
tap.options.duration  = 100;    // エフェクト表示の時間
